h2. What?

A simple, extensible performance based instrument for creating simple geometric shapes and mapping audio parameters to their physical attributes.

h2. How?

Built using "Open Frameworks":http://www.openframeworks.cc (version 0073), and written in C++.

It uses the following framework add ons:

* ofxMidi
* ofxMaxim (synthesis)
* ofxXmlSettings (serialisation)
